# HR Data Cleaning & Preprocessing with Pandas

## Overview
This project demonstrates how to clean a messy HR dataset using Python and Pandas.

## Dataset Issues
- Missing Age values
- Salary stored as text
- Missing Salary entries
- Inconsistent formatting

## Cleaning Steps
1. Handled missing Age values
2. Replaced invalid Salary values
3. Converted Salary to numeric format
4. Standardized data types
5. Exported a cleaned dataset

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Kaggle

## Output
The final dataset is ready for analysis and machine learning workflows.
